# DEPLOY_VERCEL

## Pasos
1. Entra a Vercel.
2. Pulsa **Add New** → **Project**.
3. Conecta el repositorio de GitHub donde subiste este proyecto.
4. Selecciona el repositorio.
5. Vercel detectará Next.js automáticamente.
6. Revisa que el Root Directory esté en la raíz del proyecto.
7. Pulsa **Deploy**.

## Después del deploy
- Verifica que el sitio cargue correctamente.
- Revisa el título, description y OG.
- Conecta el dominio final en Vercel si corresponde.
- Una vez conectado el dominio, vuelve a desplegar si quieres regenerar valores consistentes de metadataBase.
