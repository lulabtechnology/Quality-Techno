# ASSETS_GUIDE

## Reemplazos sugeridos

### Hero principal
- Slot visual actual: composición placeholder premium
- Reemplazo ideal: imagen editorial industrial cinematográfica, sala de bombas, planta, close-up técnico o intervención en sitio
- Proporción recomendada desktop: 1600 x 1000 px
- Proporción recomendada mobile: 1080 x 1350 px o 1080 x 1440 px si se decide crear variante

### Sobre la empresa
- Reemplazo ideal: equipo técnico, visita de planta, detalle institucional de operación
- Proporción recomendada: 1400 x 900 px

### OG image
- Archivo actual: `public/og/og-qts.svg`
- Se puede sustituir por PNG/JPG 1200 x 630 px
- Si cambias el nombre del archivo, actualiza `src/app/layout.tsx`

## Criterios visuales
- Nada de fotos genéricas de oficina corporativa aburrida
- Favorecer metal, fluidos, ingeniería, equipos, intervención técnica y atmósfera industrial premium
- Mantener coherencia con la paleta oscura + dorado + azul acero
