# INSTALL_GITHUB_WEB

## Objetivo
Subir este proyecto a GitHub Web sin usar terminal avanzada.

## Pasos
1. Crea un repositorio nuevo en GitHub.
2. Descomprime el ZIP en tu computadora.
3. En GitHub, entra al repositorio vacío.
4. Pulsa **Add file** → **Upload files**.
5. Arrastra TODO el contenido de la carpeta del proyecto.
6. Espera a que carguen todos los archivos.
7. Escribe un commit como: `feat: landing premium industrial qts fase 4`
8. Pulsa **Commit changes**.

## Validación
- Debes ver carpetas como `src`, `public`, `docs`
- Debes ver archivos como `package.json`, `next.config.mjs`, `tailwind.config.ts`
